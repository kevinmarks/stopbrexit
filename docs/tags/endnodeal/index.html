Change the default - stop a crash-out Brexit

August 21, 2019

When Parliament returns, we need to change the default from crash-out no deal Brexit to Revoking Article 50 and remaining in the EU.

There is no majority in the country or Parliament for a damaging crash-out Brexit, but there is a lot of disagreement on what we should do instead. Accordingly the first thing Parliament, or any new interim PM should do is pass a Bill to prevent an accidental Crash Out Brexit. This Bill has already been drafted - see earlier discussion by Prof. Mark Elliott, and the Bill text as as PDF (originally here).

Once this has been passed, we can then have the debate over Extension, Election, Referendum, negotiating a new Deal, secure in the knowledge that the default outcome is a safe one, not a disaster.
